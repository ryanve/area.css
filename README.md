# area.css
CSS functional `width` and `height` module

## setup

```
npm install area.css
```

```css
@import 'node_modules/area.css/area';
```

## classes

[area.css](area.css)

## example

```html
<figure class="width=60vw min-height=30vh">
  <pre class="width=6 max-width=50em height=inherit">
    width: 50%;
    max-width: 50em;
    height: inherit;
  </pre>
</figure>
```

## development

```
npm start
```
